### Project – Content Management System (CMS)

### 📝 Project Description
The project is a **WPF application** that simulates a simple **content management system** on a chosen topic.
The application supports user login, tabular display of content, and object management (adding, editing, deleting, and viewing).

### 👥 User Types
- **Admin** – can add, edit, and delete content.
- **Visitor** – can only view content.

### ⚙️ Main Features
- System login using predefined accounts (Admin / Visitor).
- Tabular content display with selection and deletion support.
- Adding a new object via a form with validation and image preview.
- RichTextBox text editor with basic options (Bold, Italic, Underline, colors, fonts, size, status bar with word count).
- Viewing or editing an existing object through a dedicated window (depending on user type).
- Serialization of users and content into **XML** files; text content stored in **RTF** files; relative paths used throughout.
